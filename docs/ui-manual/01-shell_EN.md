# 01 Shell and global actions

## Layout

| Area | Typical location | Purpose |
| --- | --- | --- |
| Primary nav | Left (wide) or top/drawer (narrow) | Home, Research, Signals, Portfolio, Settings |
| Main content | Center | Active feature page |
| Notification bell | Top or sidebar | Latest signals and alerts |
| Search / command entry | Sidebar search or shortcut | Jump to pages, find stocks, common actions |

Desktop and Web share the same information architecture; desktop-only windows follow the shipped client.

## Command palette

- Shortcut: `Cmd + K` (macOS) or `Ctrl + K` (Windows/Linux).
- Type a page name (for example Analysis, Portfolio, Signals) to navigate.
- Search stocks and open related analysis entry points.
- Some global actions (such as market review) may appear here depending on version.

## Notification bell

1. Open the bell for aggregated recent signals and alerts.
2. Click an item to deep-link into the Signal center record.
3. Unread state for signals and alerts is usually tracked separately; clearing site data may reset boundaries.
4. If one channel fails, the UI shows partial unavailability and allows retry without marking recovered items as already read.

## UI language vs report language

| Type | Affects | How to change |
| --- | --- | --- |
| UI language | Nav, buttons, settings labels | Switch in the UI (commonly zh/en), including before login when available |
| Report language | Analysis report body and some notification copy | Configure under **Settings**; independent of UI language |

Changing UI language does not change report language.

## Theme

Switch light / dark from Settings or the top bar. Preference is stored locally in the browser or desktop client.

## Habits

- After changing settings, click **Save** and wait for success before leaving the page.
- Use the command palette when you need to return to analysis quickly.
