# 03 分析工作台

路径：导航 **研究 → 分析**（或命令面板搜索「分析」）。

页面通常分为三个分段：

| 分段 | 用途 |
| --- | --- |
| 发起与批量 | 输入股票、选策略、提交任务 |
| 运行中任务 | 查看进度、失败原因 |
| 历史与对比 | 打开历史报告、趋势对比、删除 |

URL 可能带有 `segment`、`recordId` 等参数，用于恢复分段与报告。

## 发起分析

1. 输入代码，例如 `600519`、`hk00700`、`AAPL`。
2. （可选）从自选股快速选择。
3. （可选）选择策略 Skill；不选则使用系统默认。
4. （可选）切换新手 / 专业模式，或报告详细程度。
5. 点击开始分析。
6. 自动进入「运行中任务」，等待完成。
7. 完成后打开历史报告，或按完成提示进入详情。

### 批量与导入

- 支持一次提交多只时，在任务列表分别查看进度。
- **智能导入**：图片截图、CSV/Excel、剪贴板识别后，确认列表再提交。

### 代码格式提示

| 市场 | 示例 |
| --- | --- |
| A 股 | `600519`、`300750` |
| 港股 | `hk00700` |
| 美股 | `AAPL`、`BRK.B` |
| 日股 / 韩股 | `7203.T`、`005930.KS` |

## 任务进度

- 状态一般包括排队、分析中、完成、失败。
- 进行中可能显示阶段文案（行情、新闻、生成等）。
- 若显示「自动阶段」，表示按交易时段推断；最终阶段标签以报告页为准。
- 失败时阅读错误原因后再重试，避免无意义连点。

## 历史与对比

1. 打开某条历史记录查看完整报告 / Markdown。
2. 使用「历史趋势」查看同股多次结论变化。
3. 多选删除需确认，操作不可随意撤销。
4. 大盘复盘历史与个股历史分离，不要在个股列表里找复盘。

## 新手模式与专业模式

| 模式 | 体验 |
| --- | --- |
| 新手 | 更短结论、更保守风险表达 |
| 专业 | 完整字段、趋势与更多细节 |

模式偏好通常保存在本机；登出不一定清除该偏好。
