# 06 信号中心

路径：导航 **信号**（常见路由 `/signals`）。

信号中心通常包含多个 Tab：

| Tab | 作用 |
| --- | --- |
| 信号流 | 浏览结构化 AI 建议 |
| 规则 | 管理价格、涨跌幅、指标等告警规则 |
| 推送历史 | 查看触发后的通知尝试结果 |
| 再评估 / 统计 | 后验评估与汇总 |

## 信号流

1. 默认关注 **进行中（active）** 信号。
2. 可按市场、代码、动作、阶段、来源等筛选。
3. 范围控件（若有）可在全部 / 持仓 / 自选之间切换；推送历史与部分统计可能保持全局口径。
4. 打开详情查看置信度、期限、价位计划、观察条件、风险与来源报告。
5. 可将信号标记为关闭、作废或归档；终端状态一般不能直接改回 active。
6. 可对单条反馈「有用 / 无用」。

**理解**：信号是可追踪的建议记录，不是自动下单指令。

## 告警规则

1. 在 **规则** Tab 新建规则。
2. 选择类型（价格突破、涨跌幅、放量、均线/RSI/MACD、持仓风险、大盘状态等，以界面选项为准）。
3. 设定标的范围并保存、启用。
4. 如有试运行，先 dry-run 再长期启用。
5. 注意冷却状态，避免重复打扰。

## 推送历史与再评估

- 推送历史用于确认是否真正发出、各渠道结果如何。
- 再评估需按页面提示显式触发，用于回顾历史信号表现。
