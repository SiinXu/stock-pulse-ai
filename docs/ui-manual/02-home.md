# 02 首页

首页是**注意力中枢**，不是完整报告列表。

## 常见区块

| 区块 | 作用 |
| --- | --- |
| 今日焦点 | 需要优先关注的摘要 |
| 待办 | 配置缺口、未处理事项等 |
| 信号摘要 | 近期 AI 建议 / 告警概览 |
| 可展开区 | 晨报、大盘、最近分析等（展开偏好可记在本机） |

## 首次进入

1. 若提示配置不完整，优先补齐 **AI 模型** 与 **自选股**（新闻源建议配置，非绝对阻塞基础分析）。
2. 可选择「去设置」手动配置；若产品提供助手引导入口，可从首页主按钮进入。
3. 配置保存成功后再回到首页，缺口提示应减轻或消失。

## 常用操作

- 从首页进入 **研究 / 分析工作台** 开始分析。
- 点击信号摘要跳转 **信号中心**。
- 用顶栏通知铃查看未读并深链详情。

## 注意

- 带旧报告参数的链接可能被重定向到分析工作台对应分段。
- 无分析参数时，打开 `/` 应停留在首页，而不是恢复旧报告上下文。
