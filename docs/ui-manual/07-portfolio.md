# 07 持仓

路径：导航 **持仓**。

## 查看

1. 选择账户或查看全部。
2. 切换成本法（如 FIFO / 平均成本，以界面为准）。
3. 查看市值、盈亏、集中度等 KPI。
4. 阅读风险摘要（集中度、回撤、止损接近度等）。
5. 持仓行可能异步显示关联的最新 AI 信号；无信号时显示空占位属于正常。

## 记账

- 新增或删除（归档）账户。
- 录入买卖交易、现金流水、公司行动。
- 在事件列表中筛选并修正单条记录。
- 卖出超过可用数量时会被拦截。

## CSV 导入

1. 选择券商格式或通用模板。
2. 先预览（dry-run）。
3. 确认无误后再正式写入。
4. 重复提交时系统会尽量按幂等避免记两笔。

## 从持仓分析

在持仓行发起分析 → 在任务列表跟踪 → 完成后阅读报告。  
多账户持有同一标的时，按提示选择账户。
