# 11 日常工作流与界面常见问题

## 推荐工作流

### 每天约 5 分钟

1. 打开首页看焦点与待办。
2. 扫一眼信号中心的 active 信号。
3. 持仓页看风险与异常信号。
4. 对疑问股打开最近报告或问股追问一句。

### 深度研究一只股票

1. 在分析工作台提交该股（可选策略）。
2. 任务完成后按 [08 阅读报告](08-reading-reports.md) 顺序阅读。
3. 用历史趋势对比前几次结论。
4. 用问股追问失效条件与观察点。
5. 需要盯盘时在信号中心添加规则。

### 复盘自己的使用方式

1. 运行回测看历史建议表现。
2. 在信号中心查看后验与有用/无用反馈。
3. 回到设置微调模型或默认策略，而不是频繁大幅改自选制造噪音。

## 界面常见问题

**点了分析一直转圈？**  
查看「运行中任务」是否失败及原因；修复模型或网络问题后再试。

**报告语言和界面语言不一致？**  
二者独立：界面语言改导航文案，报告语言在设置中单独配置。

**信号中心是空的？**  
需要先有成功的个股分析且系统提取了信号；过旧历史不一定回填。可先对自选跑一轮分析。

**持仓有了但没有 AI 建议？**  
信号异步加载；该股无 active 信号时显示空占位。

**导入 CSV 失败？**  
先用预览检查表头与解析结果；处理超卖或重复行后再提交。

**问股串股或答非所问？**  
明确当前代码；对比时写「对比 A 和 B」。
