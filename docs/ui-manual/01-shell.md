# 01 壳层与全局操作

## 布局

| 区域 | 常见位置 | 作用 |
| --- | --- | --- |
| 主导航 | 左侧（宽屏）或顶部/抽屉（窄屏） | 首页、研究、信号、持仓、设置等 |
| 主内容区 | 中央 | 当前功能页 |
| 通知铃 | 顶栏或侧栏 | 最新信号与告警入口 |
| 搜索 / 命令入口 | 侧栏搜索或快捷键 | 跳转页面、搜股票、常用操作 |

桌面客户端与 Web 工作台的信息架构一致；个别桌面专属窗口以实际客户端为准。

## 命令面板

- 快捷键：`Cmd + K`（macOS）或 `Ctrl + K`（Windows/Linux）。
- 可输入页面名（如「分析」「持仓」「信号」）快速跳转。
- 可搜索股票并进入相关分析入口。
- 部分全局操作（如触发大盘复盘）也会出现在此，以当前版本列表为准。

## 通知铃

1. 点击铃铛查看聚合的最新信号与告警。
2. 点击条目可深链到信号中心对应记录。
3. 信号与告警的未读状态通常分开记录；清理浏览器站点数据可能重置未读边界。
4. 某一通道加载失败时，界面会提示部分不可用，可重试，不应把恢复后的条目误标为已读。

## 界面语言与报告语言

| 类型 | 影响范围 | 如何改 |
| --- | --- | --- |
| 界面语言 | 导航、按钮、设置文案等 | 登录前后均可在界面切换（常见为中/英） |
| 报告语言 | 分析报告正文、部分通知文案 | 在 **设置** 中配置；与界面语言独立 |

切换界面语言不会自动改报告语言。

## 主题

在设置或顶栏切换浅色 / 深色主题。偏好一般保存在本机浏览器或客户端本地。

## 使用习惯

- 改设置后务必点击 **保存**，看到成功提示再离开页面。
- 需要快速回到分析时，用命令面板比层层点导航更快。
