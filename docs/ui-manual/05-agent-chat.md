# 05 问股对话

路径：导航中的 **问股 / 对话**，或从个股报告进入「继续问股」。

## 基本用法

1. 确认当前标的（从报告进入时通常已带上）。
2. 用自然语言提问，例如支撑位、是否继续持有、风险点。
3. 可多轮追问；换股时明确说出新代码或名称。
4. 需要时使用导出或发送到通知渠道（若界面提供）。

## 策略

- 可在对话或分析前选择策略（趋势、成长、事件等，以产品列表为准）。
- 不选则使用默认策略。

## 使用建议

- 一次聚焦一只股票，结论更稳定。
- 对比多只时写明「对比 / vs」，减少串股。
- 问股比单次报告更耗模型额度，长对话注意用量（设置中可查看用量看板）。

## 与分析工作台的关系

- 工作台产出结构化报告与历史记录。
- 问股适合在报告基础上追问细节，不替代首次完整分析。
