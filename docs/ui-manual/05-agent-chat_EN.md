# 05 Agent chat

Path: **Chat / Ask**, or **Continue in chat** from a stock report.

## Basics

1. Confirm the active symbol (usually carried from a report).
2. Ask in natural language (support levels, hold-or-not framing, risks).
3. Continue multi-turn; when switching symbols, state the new code clearly.
4. Export or send to a notification channel when the UI offers it.

## Strategies

- Optionally pick a strategy before or during chat (trend, growth, event-driven, and others as listed).
- The default strategy applies when none is selected.

## Tips

- One symbol per focus session is more reliable.
- For comparisons, write “compare / vs” explicitly.
- Chat uses more model budget than a single report; check usage under Settings when needed.

## Versus the analysis workbench

- The workbench produces structured reports and history.
- Chat is best for follow-up questions on top of a report, not a full replacement for first-pass analysis.
