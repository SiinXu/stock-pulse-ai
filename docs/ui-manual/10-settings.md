# 10 设置（界面操作）

路径：导航 **设置 / 系统设置**。  
每改一组配置后点击 **保存**，看到成功提示再切换页面。

## AI 与模型

1. 添加模型服务（如 Anspire、AIHubMix、OpenAI 兼容、本地等）。
2. 粘贴 Key，选择或拉取模型。
3. 指定主分析模型；需要时单独指定问股 / Agent 模型。
4. 使用 **测试连接**。
5. 本地模型：在本地模型相关面板浏览、安装/注册后再选用（以客户端是否提供该面板为准）。

## 基础与自选股

- 编辑自选股列表，多个代码用逗号分隔。
- 保存后，首页与分析页的自选入口会更新。

## 数据源与新闻

- 按需填写增强数据源或搜索类 Key。
- 不配新闻源仍可能完成基础技术分析，但舆情与事件质量会下降。

## 通知

- 按渠道填写 Webhook、Bot Token、邮箱等字段。
- 使用页面上的测试推送（若有）。
- 可调整报告详略、是否单股推送等行为项（以实际开关为准）。

## 其他常用项

- 界面语言、报告语言
- 主题
- 管理员密码（若开启登录）
- 用量与成本看板
- 配置备份的导入 / 导出（桌面端尤其建议定期导出）

## 本手册不覆盖的内容

密钥申请平台账号、服务器端口、Docker 映射等见部署与完整指南，不在界面手册展开。
