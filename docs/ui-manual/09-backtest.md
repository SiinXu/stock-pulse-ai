# 09 回测

回测页用于评估**历史 AI 建议**事后表现，不是完整量化回测平台替代品。

## 操作

1. 进入回测相关页面（导航位置以当前版本为准）。
2. （可选）限定股票或分析日期范围。
3. 点击运行回测。
4. 查看结果列表与汇总指标。
5. 可下钻单股表现。

## 如何理解指标

| 概念 | 含义 |
| --- | --- |
| 方向准确率 | 涨跌方向是否与建议一致 |
| 胜率 | 有明确胜负结果时的胜负比 |
| 模拟收益 | 按规则模拟执行后的收益参考 |
| 止盈 / 止损触发率 | 计划价位是否曾触及 |

## 注意

- 过新的记录可能仍在冷却窗口内，不会纳入。
- 无结果时阅读页面诊断说明（样本不足、日期范围等）。
