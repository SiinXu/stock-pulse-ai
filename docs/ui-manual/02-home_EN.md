# 02 Home

Home is an **attention hub**, not a full report library.

## Common sections

| Section | Purpose |
| --- | --- |
| Today focus | Highest-priority summary |
| Todos | Configuration gaps and pending items |
| Signal summary | Recent AI suggestions / alerts overview |
| Expandable areas | Morning brief, market review, recent analysis (expand preference may be stored locally) |

## First visit

1. If the UI reports incomplete setup, prioritize **AI model** and **watchlist** (news sources are recommended but not always required for basic technical analysis).
2. Open **Settings** manually, or use an assistant/onboarding entry when the product exposes one.
3. After a successful save, return to Home; gap prompts should reduce or clear.

## Common actions

- Open **Research / Analysis workbench** to start analysis.
- Click the signal summary to open **Signal center**.
- Use the notification bell for unread items and deep links.

## Notes

- Legacy links with report parameters may redirect into the analysis workbench segment.
- Opening `/` without analysis parameters should stay on Home instead of restoring an old report context.
