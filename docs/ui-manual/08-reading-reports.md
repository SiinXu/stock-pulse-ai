# 08 阅读报告

打开任意**个股**历史报告时，建议按下列顺序阅读。

## 推荐顺序

1. 核心结论与操作建议（买 / 加 / 持 / 观 / 减 / 卖等）
2. 评分与趋势判断
3. 关键价位：支撑、压力、参考买卖点、止损思路
4. 风险警报与利好催化
5. 舆情、业绩与资讯摘要
6. 市场位置（题材与个股角色；A 股更常见）
7. 阶段标签与数据质量（盘前 / 盘中 / 盘后、是否降级）
8. （可选）信号归因：技术、新闻、基本面、大盘权重

## 阅读纪律

- 震荡且资金方向不清时，报告更常给出观望或持有，而不是高频翻转买卖。
- 出现「日线未完成」或数据降级时，降低对盘中结论的权重。
- 所有内容仅供研究，不构成投资建议。
